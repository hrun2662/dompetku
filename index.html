Firebase Security Check

Kenapa ini terjadi?

Karena website Anda di-host di domain baru (vercel.app), Firebase memblokir sistem login Google demi keamanan sebelum domain baru ini didaftarkan.

Cara mudah menyelesaikannya di Firebase:

  1. Buka Firebase Console Anda.
  2. Pilih proyek Anda, lalu buka menu Authentication di sebelah kiri.
  3. Klik tab Settings (atau klik Setelan).
  4. Di bawah bagian Authorized domains (Domain Resmi), klik tombol Add domain.
  5. Masukkan domain Anda: vercel.app lalu klik Add.

Setelah domain ditambahkan ke Firebase Console, halaman tinggal di-refresh dan login Google Anda akan berjalan lancar seketika!
